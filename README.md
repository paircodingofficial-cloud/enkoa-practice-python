# 파이썬 실습자료

엔코아 AI캠퍼스 「데이터 분석 & AI 머신러닝 캠프」 실습용 주피터 노트북입니다.
수업 진도에 맞춰 자료가 계속 추가됩니다.

---

## ⚠️ 가장 중요한 규칙

> ### 배포된 파일은 직접 고치지 마세요.
> ### 실습은 **`내작업/` 폴더에 복사해서** 하세요.

원본을 직접 수정하면 다음에 새 자료를 받을 때 충돌이 나서 업데이트가 막힙니다.
노트북(`.ipynb`)은 충돌을 푸는 게 아주 어렵기 때문에, 이 규칙 하나만 지키면
학기 내내 문제가 생기지 않습니다.

---

## 처음 한 번만 — 준비하기

### 1. 자료 내려받기

```bash
git clone https://github.com/paircodingofficial-cloud/enkoa-practice.git
cd enkoa-practice
```

### 2. 주피터 설치하고 실행하기

환경 구축도 실습의 일부입니다. 파이썬(3.12 이상)이 설치된 상태에서:

```bash
pip install jupyter
jupyter lab
```

> `day02`~`day04` 파이썬 기초는 **추가 라이브러리 없이** 표준 라이브러리만으로 진행합니다.
> 이후 일차부터 필요한 라이브러리는 그때그때 안내합니다.

---

## 매일 — 실습하는 방법

1. **복사합니다.** 오늘 할 파일을 `내작업/` 안으로 복사하세요.

   ```bash
   mkdir -p 내작업/day02
   cp day02_변수_제어문/*.ipynb 내작업/day02/
   ```

2. **복사본을 엽니다.** 주피터에서 `내작업/day02/` 안의 파일을 열어 실습합니다.

3. **원본은 그대로 둡니다.** 참고용으로만 보세요.

---

## 새 자료 받기

수업이 진행되면 새 폴더가 추가됩니다. 받을 때는:

```bash
./업데이트.sh
```

처음 한 번은 실행 권한이 필요할 수 있습니다:

```bash
chmod +x 업데이트.sh
```

이 스크립트는 **`내작업/` 폴더를 절대 건드리지 않습니다.** 배포 원본만 최신으로
맞춥니다. 혹시 원본을 고쳤더라도 `.백업/` 폴더로 옮겨 두니 잃어버리지 않습니다.

---

## 폴더 구성

```
enkoa-practice/
├── day02_변수_제어문/          ← 배포 자료 (읽기 전용으로 생각하세요)
│   ├── 교안_01_변수_자료형_입출력_연산자.ipynb
│   ├── 교안_02_조건문_반복문.ipynb
│   ├── 과제_LV1_기초.ipynb
│   ├── 과제_LV2_응용.ipynb
│   └── 과제_LV3_통합.ipynb
├── 내작업/                     ← 여기서 실습하세요 (GitHub에 안 올라감)
└── 업데이트.sh                 ← 새 자료 받기
```

### 파일 종류

| 종류 | 내용 |
|---|---|
| **교안** | 수업 중 함께 진행합니다. 🖐️ 표시된 셀은 직접 따라 쳐 보세요. |
| **과제 LV1 (기초)** | 배운 문법을 하나씩 확인하는 기본 문제 |
| **과제 LV2 (응용)** | 여러 개념을 엮어 푸는 문제 |
| **과제 LV3 (통합)** | 단원 전체를 활용하는 통합 문제 |

과제에는 `# [자가채점]` 셀이 있습니다. 답을 작성한 뒤 실행하면 스스로 확인할 수 있습니다.

## 자주 겪는 문제

**`./업데이트.sh` 가 실행되지 않아요**
→ `chmod +x 업데이트.sh` 를 한 번 실행하세요.

**`git pull` 을 직접 했더니 에러가 나요**
→ `./업데이트.sh` 를 대신 쓰세요. 원본 수정분을 알아서 정리해 줍니다.

**노트북을 실행했더니 파일이 바뀌었다고 나와요**
→ 정상입니다. 셀을 실행하면 결과가 파일에 기록됩니다. `./업데이트.sh` 가 처리하니
   신경 쓰지 마세요. 다만 **앞으로는 `내작업/` 에 복사해서** 실행하세요.
